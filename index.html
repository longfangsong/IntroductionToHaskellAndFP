What's the difference

  • 没有变量，只有常量
  • 没有传统意义上的逻辑判断
  • 没有循环
  • 没有子程序，只有函数
  • 函数是”一等公民”，每个函数都是一个值，有其类型，能作为其他函数的参数
  • 惰性求值
  • 静态强类型
  • Int
  • Integer
  • Float
  • Double
  • Char
  • Bool
